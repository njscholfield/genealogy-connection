About This Project

This site is an archived, static version of a demo site I built for a class at Pitt in 2016. The original version had a Node.js backend which allowed you to create an account and link it to the now discontinued 23andMe API to pull in basic information from your profile.

You can see the original code on GitHub if you are interested. I was originally just going to make this project private but I decided to make it into this static site instead since I still like the design, even 4 years later.

If you want to see more recent projects or mine, check out my projects page.
